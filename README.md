**Front-end Immersion – Alura**

This project was developed during the Front-end Immersion by Alura, at the beginning of my Systems Development course.
The goal of this project was to recreate the main interface of Spotify, focusing on the page where albums, artists, and music categories are displayed.
This project was created as a practical exercise to apply basic front-end development concepts.

**🚀 Project Goal**
The objective of this project was to practice building web interfaces using HTML, CSS, and JavaScript, recreating a layout inspired by Spotify’s homepage.
During development, the focus was on:
Structuring the page with HTML
Styling and layout using CSS
Organizing artist and album cards
Practicing interface design and layout organization

**🛠 Technologies Used**
HTML5
CSS3
JavaScript

**🎯 What I Learned**
During this project I practiced:
Structuring web pages
Styling visual components
Creating layouts inspired by real applications
Organizing elements using cards and grids

**⚠️ Disclaimer**
This project was developed for educational purposes only as part of the Alura front-end immersion.
The interface is inspired by Spotify, but it has no official connection with the platform.
